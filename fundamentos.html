Dimensiones clave de calidad (según ISO 9126/25000):

  • Funcionalidad: El software cumple con las funciones para las que fue diseñado.
  • Fiabilidad: La capacidad del software para realizar su función de manera consistente.
  • Usabilidad: La facilidad con la que los usuarios pueden interactuar con el software.
  • Eficiencia: El software usa los recursos de manera óptima.
  • Mantenibilidad: Qué tan fácil es modificar el software para corregir errores o adaptarlo a nuevos requerimientos.li>
  • Portabilidad: La facilidad con la que el software puede ser adaptado a diferentes entornos.
